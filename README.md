# OKX Quant Workbench

当前版本：`v0.3.14`

这是一个面向 OKX 的桌面交易工作台，当前已经包含：

- 策略运行与多会话管理
- 账户持仓、历史成交、历史仓位
- 期权持仓保护与回放模拟
- 多币种信号监控与邮件通知
- Deribit 波动率指数与波动率监控
- 回测、矩阵对比、历史报告持久化
- 无限下单工作台
- 期权策略计算器

## 1. 策略工作台

主界面支持同时运行多个策略会话，当前已接入：

- `EMA 动态委托`
- `EMA 穿越市价`
- `4H EMA5/EMA8 金叉死叉`

支持：

- 多策略同时运行
- 会话列表查看与停止
- 策略参数详情查看
- 多 API 账户切换

## 2. 账户持仓

主界面和持仓大窗提供接近 OKX Windows 客户端的持仓视图，支持：

- 当前持仓树形分组
- 风险单元 / 到期日分组
- 历史成交
- 历史仓位
- 列显示控制
- 筛选与统计
- 账户信息弹窗

持仓大窗支持：

- 折叠持仓详情
- 折叠历史成交详情
- 折叠历史仓位详情
- 默认详情区域关闭
- 历史成交默认读取 `500` 条
- 支持继续增加历史成交条数
- 历史成交支持类型 / 方向 / 搜索筛选
- 历史成交会补充到期行权 / 交割类记录

## 3. 期权持仓保护

支持针对已有期权仓位设置本地止盈止损保护。

支持方式：

- 按期权标记价格触发
- 按现货价格触发
- 设定价格报单
- 标记价格加减滑点报单

内置风控：

- 网络异常持续重试
- 3 分钟限频异常邮件
- 防重复平仓
- 反向仓位熔断
- 价格异常拦截

价格异常拦截会参考：

- 现货价格
- 行权价
- 开仓均价
- bid / ask / last

若报单价格明显异常：

- 不自动报单
- 任务进入异常
- 发邮件提醒
- 等待人工处理

## 4. 多币种信号监控

支持收线后监控，不做无意义秒轮询。

当前支持的信号：

- `EMA21/55 金叉死叉`
- `EMA55 斜率改变`
- `EMA55 突破`
- `K线形态信号`

特点：

- 任务模式运行
- 关闭窗口不停止任务
- 最近触发信号列表
- 实时诊断区
- 信号邮件通知

## 5. Deribit 波动率工具

当前包含两部分。

### Deribit 波动率指数窗口

- 支持 `BTC / ETH`
- 支持 `1小时 / 4小时 / 12小时 / 1天`
- 默认最多查看 `1000` 根 K 线
- 支持放大、缩小、拖动、重置视图

说明：

- Deribit 官方接口没有原生 `4小时`
- 程序会用 `1小时` 数据本地聚合成 `4小时`

### 波动率监控

当前支持的明显信号：

- 连涨后大阴反转
- 连跌后大阳反转
- 窄幅后大阳突破
- 窄幅后大阴突破
- 箱体向上突破
- 箱体向下突破
- `EMA34` 转强
- `EMA34` 转弱

支持：

- 收线后检测
- 邮件提醒
- 实时诊断
- 任务列表

## 6. 回测系统

回测入口：

- `工具 -> 打开回测窗口`
- `工具 -> 打开回测对比总览`

当前支持：

- `5分钟`
- `15分钟`
- `1小时`
- `4小时`

能力包括：

- 单次最多 `10000` 根已收盘 K 线
- 本地历史 K 线缓存
- 自动补最新历史数据
- 支持按时间段回测
- 自动补足指标预热和前置 K 线
- 前 `200` 根仅用于指标预热，不参与交易
- K 线图放大、缩小、拖动
- 悬停显示时间、价格、EMA
- 资金曲线
- 当前报告
- 回测对比
- 矩阵对比
- 周期统计
- 历史回测持久化

回测成本模型支持：

- Maker 手续费
- Taker 手续费
- 滑点
- 资金费率

ATR 批量矩阵规则：

- `止损 1 ATR -> 止盈 1 / 2 / 3 ATR`
- `止损 1.5 ATR -> 止盈 1.5 / 3 / 4.5 ATR`
- `止损 2 ATR -> 止盈 2 / 4 / 6 ATR`

## 7. 无限下单

入口：

- `工具 -> 打开无限下单`

这是一个单合约弹出式交易工作台。

特点：

- 一个窗口只服务一个合约
- 只要存在活动任务，窗口就锁定当前合约
- 关闭窗口前必须停止任务并撤掉活动委托

第一版已支持：

### 手工下单

- 限价
- 最优价
- 市价（激进 IOC）
- IOC
- FOK
- Post Only

最优价逻辑：

- `buy` 挂 `买一`
- `sell` 挂 `卖一`

期权数量口径：

- 当前合约类型为 `OPTION` 时，手工下单数量按“币数”输入
- 程序会按合约面值自动换算成张数后再下单

### 条件单

- 上穿 / 下穿触发
- 当前合约 / 自定义标的触发
- 触发后限价或激进 IOC

### 止盈止损

- 可针对当前合约单独启动
- 平仓逻辑复用严格风控

### 点击报价网格

这不是传统铺多层网格，而是“点击报价后的往返挂单链”。

支持：

- 同一合约下多条网格任务并存
- 买成后按“多单参数”挂反向卖单
- 卖成后按“空单参数”挂反向买单
- 多单参数、空单参数可不同

### 规则盘口

规则盘口支持：

- 按交易所盘口量显示买卖量
- 自定义价格筛选步长
- 当前价 / 买一 / 卖一高亮
- 点击价格带入
- 网格开关开启后，点击买卖列可直接创建网格任务

### 任务持久化

若软件异常关闭：

- 网格任务
- 条件单
- 止盈止损任务

会在下次打开时恢复显示，并允许再次启动。

## 8. 期权策略计算器

入口：

- `工具 -> 打开期权策略计算器`

支持：

- 读取期权系列并显示认购 / 认沽链
- 导入到期持仓
- 导入系列持仓
- 保存 / 加载 / 删除策略
- 组合公式线性表达式
- 到期盈亏图
- 组合 K 线
- 策略腿分组查看
- Greeks 汇总与组合摘要

## 9. 本地文件

运行过程中会在本地生成这些文件：

- `.okx_quant_credentials.json`：API 凭证
- `.okx_quant_settings.json`：程序设置
- `.okx_quant_backtest_history.json`：回测历史
- `.okx_quant_smart_order_tasks.json`：无限下单任务快照
- `.okx_quant_smart_order_favorites.json`：无限下单自选合约
- `.okx_quant_candle_cache/`：本地 K 线缓存

更新服务器时，建议至少保留：

- `.okx_quant_credentials.json`
- `.okx_quant_settings.json`

如果希望无缝续用，再保留：

- `.okx_quant_backtest_history.json`
- `.okx_quant_smart_order_tasks.json`
- `.okx_quant_smart_order_favorites.json`
- `.okx_quant_candle_cache/`
