## 策略说明

### EMA 动态委托

- 选择交易对、K 线周期、`EMA 周期`
- 设置 `趋势EMA周期`
- 设置 `ATR 周期 / 止损 ATR 倍数 / 止盈 ATR 倍数`
- 设置 `风险金`
- 选择方向：`只做多` 或 `只做空`
- 点击启动后，程序会以上一根已收盘 K 线的 EMA 作为开仓价格挂限价委托
- 每一根新 K 线确认后，先撤掉旧挂单，再按最新的上一根 EMA 重新挂单
- 仓位按风险金自动计算：`开仓数量 = 风险金 / abs(开仓价格 - 止损价格)`
- 趋势过滤支持：
  - 价格位于趋势 EMA 上方才允许做多
  - 价格位于趋势 EMA 下方才允许做空
  - 短 EMA 在趋势 EMA 下方时不做多
  - 短 EMA 在趋势 EMA 上方时不做空

适用场景：

- 上升趋势中的回调做多
- 下降趋势中的反弹做空

### EMA 穿越市价

- 最近一根已收盘 K 线上穿 EMA 做多，下穿 EMA 做空
- 做多止损：信号 K 线最低价减去 `1 ATR`
- 做空止损：信号 K 线最高价加上 `1 ATR`
- 仓位同样按风险金自动计算
- 也支持趋势 EMA 过滤

## 持仓与保护

- 主界面支持账户持仓查看与自动刷新
- `持仓大窗` 支持：
  - 当前持仓树状查看
  - 历史成交
  - 历史仓位
  - 列显示/隐藏
  - 筛选与统计
  - 期权快捷搜索
- 期权持仓保护支持：
  - 选中期权仓位后设置止盈止损
  - 根据期权价格或现货价格触发
  - 本地持续监控任务
  - 保护回放模拟

## 多币种信号监控

- 支持默认监控：
  - `BTCUSDT`
  - `ETHUSDT`
  - `SOLUSDT`
  - `BNBUSDT`
- 支持信号：
  - `EMA21/55 金叉死叉`
  - `EMA55 斜率改变`
  - `EMA55 突破`
  - `K线形态信号`
- 监控逻辑按新 K 线收线后执行
- 支持信号邮件、诊断日志、任务列表

## 回测

- 菜单 `工具 > 打开回测窗口`
- 支持周期：
  - `5分钟`
  - `15分钟`
  - `1小时`
  - `4小时`
- 回测窗口支持：
  - K 线图、EMA、趋势 EMA
  - 止盈止损触发点
  - 交易明细
  - 当前报告
  - 回测对比
  - 矩阵对比
- 支持 ATR 批量参数测试：
  - `止损 1 ATR -> 止盈 1 / 2 / 3 ATR`
  - `止损 1.5 ATR -> 止盈 1.5 / 3 / 4.5 ATR`
  - `止损 2 ATR -> 止盈 2 / 4 / 6 ATR`
- 支持单独的 `策略回测对比总览`
- 关闭程序后仍会保留历史回测记录

## 项目结构

- `main.py`：启动桌面程序
- `okx_quant/ui.py`：主工作台界面
- `okx_quant/backtest_ui.py`：回测窗口与对比总览
- `okx_quant/backtest.py`：回测逻辑与报告计算
- `okx_quant/engine.py`：实盘策略执行逻辑
- `okx_quant/okx_client.py`：OKX REST API 封装
- `okx_quant/position_protection.py`：期权持仓保护
- `okx_quant/protection_replay_ui.py`：保护回放窗口
- `okx_quant/signal_monitor.py`：多币种信号监控
- `okx_quant/signal_monitor_ui.py`：信号监控窗口
- `okx_quant/candle_cache.py`：本地 K 线缓存
- `okx_quant/strategies/ema_dynamic.py`：EMA 动态委托策略
- `okx_quant/strategies/ema_atr.py`：EMA 穿越市价策略
- `okx_quant/strategy_catalog.py`：策略目录与说明
- `tests/`：核心逻辑测试
