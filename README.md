# OKX 策略工作台

当前版本：`v0.3.17`

这是一个面向 OKX 的桌面交易工作台，围绕“监控、交易、保护、回测、分析”构建。当前项目已经包含：

- 策略运行与多会话管理
- 账户持仓、历史成交、历史仓位
- 期权持仓保护与回放模拟
- 多币种信号监控与邮件通知
- Deribit 波动率指数查看与波动率监控
- 回测、矩阵对比、历史报告持久化
- 无限下单 / 条件单 / 网格任务
- 期权策略计算器与图表分析

## 1. 主界面与策略工作台

主界面支持同时运行多个策略会话，当前已接入：

- `EMA 动态委托`
- `EMA 穿越市价`
- `4H EMA5 / EMA8 金叉死叉`

主界面能力包括：

- 多策略同时运行
- 会话列表查看与停止
- 多 API 账户切换
- 账户持仓区优先显示
- 主界面不再显示选中持仓详情，重点聚焦策略与持仓

## 2. 账户持仓与历史数据

主界面和持仓大窗提供接近 OKX Windows 客户端的持仓视图，支持：

- 当前持仓树形分组
- 风险单元 / 到期日分组
- 历史成交
- 历史仓位
- 账户信息弹窗
- 自定义列显示
- 搜索、筛选、摘要统计

当前持仓与持仓大窗的增强点：

- 分组行支持显示 `持仓个数 + 持仓数量汇总`
- 摘要区支持显示：
  - `买购数量`
  - `卖购数量`
  - `买沽数量`
  - `卖沽数量`
- `标记价 / 标记≈USDT`
- `开仓价 / 开仓≈USDT`
- `浮盈亏 / 浮盈≈USDT`
- `Theta / Theta≈USDT`
- 大窗里 `≈USDT` 列宽已做紧凑化处理

历史成交当前支持：

- 默认读取 `500` 条
- 单独 `刷新历史成交`
- `增加100条`，之后每次 `增加200条`
- 按 `类型 / 方向 / 搜索` 筛选
- `带入合约`
- `带入到期前缀`
- 补充到期被行权 / 交割类记录

显示口径：

- 行权 / 交割成交价按 `2` 位小数显示
- `已实现盈亏` 按交易币种精度显示

## 3. 期权持仓保护

支持针对已有期权仓位设置本地止盈止损保护。

支持方式：

- 按期权标记价格触发
- 按现货价格触发
- 设定价格报单
- 标记价格加减滑点报单

内置风控：

- 网络异常持续重试
- 3 分钟限频异常邮件
- 防重复平仓
- 反向仓位熔断
- 价格异常拦截

价格拦截会参考：

- 现货价格
- 行权价
- 开仓均价
- bid / ask / last

如果报单价格明显异常：

- 不自动报单
- 任务进入异常
- 发邮件提醒
- 等待人工处理

## 4. 多币种信号监控

支持按收线后检测信号，不做无意义秒级轮询。

当前支持的信号包括：

- `EMA21/55 金叉死叉`
- `EMA55 斜率改变`
- `EMA55 突破`
- `K线形态信号`

监控窗口支持：

- 监控任务
- 最近触发信号
- 监控日志
- 实时诊断
- 整页纵向滚动，适应不同分辨率
- 历史信号图预览 / 放大查看

通知方式：

- 邮件通知
- 实时诊断输出

## 5. Deribit 波动率工具

项目内包含两块 Deribit 波动率能力。

### 5.1 波动率指数窗口

支持：

- `BTC / ETH`
- `1小时 / 4小时 / 12小时 / 1天`
- 默认最多查看 `1000` 根 K 线
- 鼠标滚轮缩放
- 左键拖动
- 双击重置视图

说明：

- Deribit 官方接口没有原生 `4小时`
- 程序会用 `1小时` 数据本地聚合成 `4小时`

### 5.2 波动率监控

当前支持的明显信号包括：

- 连涨后大阴反转
- 连跌后大阳反转
- 窄幅向上突破
- 窄幅向下突破
- 箱体向上突破
- 箱体向下突破
- `EMA34` 转强
- `EMA34` 转弱

窗口支持：

- 监控任务
- 最近触发信号
- 监控日志
- 实时诊断
- 整页纵向滚动，适应不同分辨率

## 6. 回测系统

入口：

- `工具 -> 打开回测窗口`
- `工具 -> 打开回测对比总览`

当前支持周期：

- `5分钟`
- `15分钟`
- `1小时`
- `4小时`

回测能力包括：

- 单次最多 `10000` 根已收盘 K 线
- 本地历史 K 线缓存
- 自动补最新历史数据
- 支持按时间段回测
- 自动补足指标预热和前置 K 线
- 前 `200` 根仅用于预热，不参与交易
- K 线图支持缩放、拖动、十字光标、悬停详情
- 资金曲线 / 净值相关展示
- 当前报告
- 回测对比
- 矩阵对比
- 参数热力图
- 周期统计
- 历史回测持久化

回测成本模型支持：

- Maker 手续费
- Taker 手续费
- 滑点
- 资金费率

ATR 批量矩阵规则：

- `止损 1 ATR -> 止盈 1 / 2 / 3 ATR`
- `止损 1.5 ATR -> 止盈 1.5 / 3 / 4.5 ATR`
- `止损 2 ATR -> 止盈 2 / 4 / 6 ATR`

## 7. 无限下单

入口：

- `工具 -> 打开无限下单`

这是一个单合约弹出式交易工作台。

特点：

- 一个窗口只服务一个合约
- 有活动任务时锁定当前合约
- 关闭窗口前必须停止任务并撤掉活动委托

### 7.1 手工下单

支持：

- 限价
- 最优价
- 市价（激进 IOC）
- IOC
- FOK
- Post Only

最优价逻辑：

- `buy` 挂 `买一`
- `sell` 挂 `卖一`

期权数量口径：

- 当前合约类型为 `OPTION` 时，数量按“币数”输入
- 程序会按合约面值自动换算成张数
- 仓位限制也按币数输入和显示，内部自动换算成张数

### 7.2 条件单

支持：

- 上穿 / 下穿触发
- 当前合约 / 自定义标的触发
- 触发后限价或激进 IOC

### 7.3 止盈止损

支持对当前合约单独启动本地止盈止损任务，并复用期权保护的严谨风控。

### 7.4 点击报价网格

这里不是传统铺多层静态网格，而是“点击报价后的往返挂单链”。

支持：

- 同一合约下多条网格任务并存
- 买成后按“多单参数”挂反向卖单
- 卖成后按“空单参数”挂反向买单
- 多单参数、空单参数可不同

### 7.5 规则盘口

支持：

- 显示交易所盘口买卖量
- 自定义价格筛选步长
- 当前价 / 买一 / 卖一高亮
- 点击价格带入
- 网格模式开启后，点击买卖列直接建网格任务

### 7.6 任务持久化

若软件异常关闭：

- 网格任务
- 条件单
- 止盈止损任务

会在下次打开时恢复显示，并允许再次启动。

## 8. 期权策略计算器

入口：

- `工具 -> 打开期权策略计算器`

支持：

- 读取期权链并显示认购 / 认沽报价
- 导入到期持仓
- 导入系列持仓
- 保存 / 加载 / 删除策略
- 组合公式线性表达式
- 到期盈亏图
- 组合 K 线
- 策略腿分组查看
- 图表大窗
- Greeks 汇总与组合摘要

### 8.1 策略腿

当前策略腿表格支持：

- 持仓价
- 标记价
- Delta
- Gamma
- Theta
- Vega

其中：

- `持仓价` 可手工修改，用于分析真实建仓成本
- `标记价` 用于当前市值与现价参考
- 刷新报价不会覆盖持仓价

### 8.2 组合 K 线

默认值：

- `K线周期 = 1H`
- `K线数量 = 1000`
- `图表币种 = 结算币`

特点：

- 组合 K 线采用 OKX 期权标记价格组合
- 支持缩放、拖动、十字光标
- 支持 `只影响组合K线`
- 支持 `消除影线`
  - 开启后只按 `open / close` 计算高低
- 颜色规则为：
  - `绿涨`
  - `红跌`

### 8.3 图表大窗

图表大窗包含：

- 到期盈亏图
- 组合K线
- 波动率K线
- 叠加对比

特点：

- 到期盈亏图支持 hover 跟随线
- 组合K线支持缩放、拖动、十字光标
- 波动率K线周期与组合K线一致
- `叠加对比` 改为上下双图
  - 上图：组合K线
  - 下图：波动率K线
- 上下双图共用时间轴
- 十字光标上下图联动

## 9. 本地文件

运行过程中会在本地生成这些文件：

- `.okx_quant_credentials.json`：API 凭证
- `.okx_quant_settings.json`：程序设置
- `.okx_quant_backtest_history.json`：回测历史
- `.okx_quant_smart_order_tasks.json`：无限下单任务快照
- `.okx_quant_smart_order_favorites.json`：无限下单自选合约
- `.okx_quant_candle_cache/`：本地 K 线缓存

更新服务器时，建议至少保留：

- `.okx_quant_credentials.json`
- `.okx_quant_settings.json`

如需无缝续用，再保留：

- `.okx_quant_backtest_history.json`
- `.okx_quant_smart_order_tasks.json`
- `.okx_quant_smart_order_favorites.json`
- `.okx_quant_candle_cache/`
